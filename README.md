# Simple Tabs Component

A basic frontend implementation of a tab navigation component using HTML, CSS, and vanilla JavaScript. 

## Features
- Toggle between different content views by clicking on the tabs.
- Clean and basic UI styling using CSS.
- Extremely simple, beginner-friendly JavaScript logic.

## Source Reference
This project is based on the idea from: 
[roadmap.sh - Simple Tabs](https://roadmap.sh/projects/simple-tabs)

## Getting Started
To view this project, simply open the `index.html` file in any modern web browser. No extensions, build steps, or installations are required.
